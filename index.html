VizBuilts 360 Mobile Test Viewer

This viewer is meant to test whether YouTube's embedded 360 player will honor mobile touch/gyro interaction inside a wrapper page.

Example URL:

?video=zxZBpxrye3o&t=2.7&yaw=176&pitch=0&roll=0&fov=80

This version adds iframe permissions for accelerometer, gyroscope, fullscreen, autoplay, encrypted-media, and picture-in-picture.

If mobile still shows flat/equirectangular video, that likely means YouTube is not enabling its mobile embedded 360 renderer for this browser/session.

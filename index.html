VizBuilts 360 Wrapper

This single file has two branches:

  • Desktop Mode: preserves the stable V2 YouTube iframe behavior for drag-to-look.
  • Mobile Mode: adds iframe permissions and an optional motion-permission button without changing desktop handling.

Supported URL format:

https://vizbuilts.github.io/vr360/?video=VIDEO_ID&t=18&yaw=90&pitch=-7&roll=0&fov=80

Also accepted:

  • v=VIDEO_ID instead of video=VIDEO_ID
  • time=18 instead of t=18
  • Full YouTube URL passed as url=https://youtu.be/VIDEO_ID?t=18s

YouTube reliably supports time seeking through the player API. Spherical view restoration depends on the video being a valid 360 video and the browser/player honoring setSphericalProperties().

Mobile YouTube iframe 360 interaction can still be inconsistent. If mobile does not allow drag/gyro interaction, use the Open YouTube button or move to a Three.js self-hosted 360 video viewer for reliable field/mobile use.
